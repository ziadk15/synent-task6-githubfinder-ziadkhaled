# GitHub Profile Finder

##  Project Overview
This is a simple web application that allows users to search for any GitHub profile using a username and display real-time data using the GitHub API.

The project was built as part of the Synent Technologies Web Development Internship Program (Task 6).

---

##  Features
- Search GitHub users by username
- Display user profile information
- Show avatar image, name, bio
- Display public repositories count
- Display followers and following
- Loading state while fetching data
- Error handling for invalid usernames
- Responsive design for mobile and desktop

---

##  Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API
- GitHub REST API

---

##  API Used
GitHub Users API:
https://api.github.com/users/{username}

---

##  How It Works
1. User enters a GitHub username
2. Clicks on Search button
3. The app sends a request to GitHub API
4. Data is fetched and displayed dynamically
5. Loading state is shown during request
6. Error message appears if user is not found
